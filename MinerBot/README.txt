Before starting program, you need to edit "config.cfg" file and put your browser session data of gameminer.net
Login to your profile via steam on gameminer.net, then copy your cookies '_xsrf' and 'token' to "config.cfg".
To find out your  actual browser version visit "http://whatsmyuseragent.com/".
If you did everything correctly - start GM.exe and enjoy :)

While program runs, it will automatically start searching giveaways with price 0 every 5 min!

You can check giveaways that program enters just by clicking on links that starts with 'Entered' or 'Conditions'
